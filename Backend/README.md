# SmartRoad AI - Road Damage Detection System

## Overview
AI-Based Road Damage Detection System using ESP32, Ultrasonic Sensor, GPS, and YOLO AI.

## Features
- Bluetooth connection to ESP32
- Real-time pothole detection
- GPS location tracking
- Camera capture with AI confirmation
- Detection history with map view
- Email/SMS alerts

## Setup Instructions

### Prerequisites
- Flutter SDK 3.0+
- Android Studio / VS Code
- Android SDK 21+
- iOS 12+ (for iOS builds)

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

### Hardware Requirements
- ESP32 WROOM
- HC-SR04 Ultrasonic Sensor
- NEO-6M GPS Module
- Android Smartphone
